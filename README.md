# gethashToRGB
ジオハッシュ文字列をよろしくな色にします。

usage
```
# load script tag or import geohash_to_rgb.js

var geohash = 'xn1rc';

# 適当な色
geohash.geohashToRGB();
# アタック25な色(透過)
geohash.geohashToAttack25(0.5);

```
